# Markdown Printer

A paged markdown previewer that brings markdown to real-world paper. Write in markdown, preview with pagination, and print or export to PDF — exactly as it will appear on paper.

## Features

- **Split-pane editor** — CodeMirror 6 editor with live Markdown preview
- **PDF export** — Browser-native PDF generation.
- **Multi-tab management** — Create, rename, close, and switch between documents
- **Page sizing** — A4, Letter, Legal, A3, A5, Tabloid with accurate page breaks
- **Font selection** — Google Fonts integration (Open Sans, Roboto, Montserrat, Inter, Lora, Lato, Source Code Pro)
